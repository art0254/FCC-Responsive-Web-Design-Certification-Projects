Introduction

Responsive Website คือ เว็บไซต์ที่สามารถรองรับการแสดงผลตามขนาดหน้าจอ ของอุปกรณ์ทุกชนิด โดยเว็บไซต์สามารถตรวจจับขนาดของหน้าจอ และปรับขนาด Layout ให้เหมาะสม กับอุปกรณนั้นๆ ทําให้ผู้ใช้ชมเนื้อหาได้อย่างมประสิทธภิาพ

ตัวอย่างเว็บแบบ Responsive Website เว็บไซต์ที่เป็น Responsive Website จะสามารถรองรับหน้าจอได้หลากหลาย เช่นที่หน้าจอคอมพิวเตอร์จะแสดงทั้ง 3 คอลัม แต่หน้าจอขนาดเล็กเช่นโทรศัพท์จะแสดงเพียงแค่ 1 คอมลัม

Meta Tag

เพื่อให้เว็บไซต์ที่สร้างขึ้นสามารถปรับย่อและขยายขนาดตามหน้าจอได้ จำเป็นต้องเพิ่ม Tag Meta ไว้ภายใน Tag head ของไฟล์ HTML

<meta name="viewport" content="width=device-width, initial-scale=1.0">
Grid System

เพื่อให้รองรับการแสดงผลในหน้าจอที่หลากหลาย Bootstrp ได้มีคลาสสำหรับจัดการ คือระบบ Grid System ซึ่งจะแบ่งหน้าจอเป็น 5 ขนาดได้แก่

  • Extra Small ขนาดเล็กมาก เช่น โทรศัพท์หรือSmart Watch
  • Small ขนาดเล็ก เช่น โทรศัพท์
  • Medium ขนาดกลาง เช่น Tablet
  • Large ขนาดใหญ่ เช่น หน้าจอคอมพิวเตอร์
  • Extra large ขนาดใหญ่มาก เช่น หน้าจอคอมพิวเตอร์หรือโทรทัศน์
Row & Container

เพื่อให้ใช้งานร่วมกับระบบ Grid System สามารถใช้คลาส row และคลาส container โดยคุณสมบัติของคลาส row จะมี ความกว้างเท่ากับ 100% ของหน้าจอ คือ ชิดทั้งซ้ายและขวาและมีทั้งหมด 12 คอลัมเพื่อใช้งานร่วมกับระบบ Grid System ส่วน class container จะมีคุณสมบัติ กว้างมากที่สุด (max-width) และมีการเว้นขอบซ้าย และขวาเล็กน้อย

ตัวอย่าง code การใช้งานคลาส row และ

<div class="container">
<div class="row"></div>
</div>
Row & Grid

จะใช้ระบบ Grid System เพื่อแบ่งหน้าจอให้แสดงผลตามขนาดที่แตกต่างกัน โดยภายใน 1 row จะประกอบด้วย 12 คอลัมน์ ซึ่งสามารถใช้คลาสจากตาราง Grid System มากำหนดได้ว่าที่ ขนาดหน้าจอแบบใดจะให้แสดงผลกี่คอลัมน์ โดยขนาดหน้าจอที่มีความเล็กกว่าที่กำหนดไว้จะแสดง เต็ม 12 คอลัมน์

ตัวอย่างการใช้ col โดยเราจะ แทรกสีพื้นหลังลงในข้อความด้วย stye เพื่อให้เห็นความแตกต่าง โดยเป็นสีฟ้าและสีเขียว

<div class="row">
<div style="background-color: blue;" class="col-md-4">Hello</div>
<div style="background-color: green;" class="col-md-8">Good Moring</div>
</div>

ในตัวอย่างนี้จะให้หน้าจอขนาดกลางขึ้นไปที่ Hello ใช้จำนวน 4 คอลัมน์ และ Good morning จะใช้ 8 คอลัมน์ ส่วนหน้าจอขนาดที่มีขนาดเล็กกว่าหน้าจอขนาดกลางระบบจะ แสดงผลอย่างละ 12 คอลัมน์ให้อัตโนมัติ

ให้ทดลองย่อบราวเซอร์เพื่อดูความแตกต่าง

Image

คลาสสำหรับจัดการรูปภาพ

  • img-fluid : สำหรับให้ภาพขยายตามขนาดของหน้าจอ
  • rounded : แทรกกรอบที่มีความโค้ง
  • rounded-circle : แทรกกรอบวงกลม
  • img-thumbnail : กรอบสี่เหลี่ยม และเป็นภาพสำหรับ thumbnail
<img src="image/picture-1.jpg" class="img-fluid rounded">
<img src="image/picture-1.jpg" class="img-fluid rounded-circle">
<img src="image/picture-1.jpg" class="img-fluid img-thumbnail">
Heading

เป็นคลาสสำหรับจัดการแสดงผลรูปแบบหัวข้อตัวอักษร มีรูปแบบและขนาดดังตาราง โดยสามารถใช้คลาสของ Bootstrap เพื่อจัดการเปลี่ยนขนาดได้โดยไม่ต้องเขียน Tag ของ h1-h6 ของภาษาhtml และได้รับความเป็น responsive

<p class="h1">Hello</p>
<p class="h2">Hello</p>
<p class="h3">Hello</p>
<p class="h4">Hello</p>
<p class="h5">Hello</p>
<p class="h6">Hello</p>
Text Align

เป็นคลาสสำหรับกำหนดตำแหน่งของตัวหนังสือทางแนวนอนโดยสามารถจัดชิดซ้าย กลาง และชิดขวาเพื่อเพิ่มความสะดวกและรวดเร็วในการใช้งาน

  • text-left จัดตัวหนังสือชิดซ้าย
  • text-center จัดตัวหนังสืออยู่กลาง
  • text-right จัดตัวหนังสือชิดขวา
<p class="text-left">Left</p>
<p class="text-center">Center</p>
<p class="text-right">Right</p>
